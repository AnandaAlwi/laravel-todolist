# Laravel To Do List
Ini adalah aplikasi Laravel sederhana untuk mencatat daftar tugas harian. Aplikasi ini memungkinkan pengguna menambahkan, mengedit, dan menghapus tugas sesuai kebutuhan. Dibuat sebagai latihan pengembangan aplikasi berbasis Laravel. Proyek ini cocok untuk belajar dasar-dasar Laravel dan manajemen tugas.
